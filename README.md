# ⛅ WeatherNow

App del clima en tiempo real que consume la API de OpenWeatherMap. Desarrollada como proyecto de portafolio.

🌐 **Demo en vivo:** [nahuelprogram17.github.io/weather-app](https://nahuelprogram17.github.io/weather-app)

## Tecnologías utilizadas

- HTML5 semántico
- CSS3 (variables, Grid, Flexbox, animaciones)
- JavaScript vanilla (Fetch API, async/await, manejo de errores)
- OpenWeatherMap API (datos en tiempo real)
- Git & GitHub para control de versiones
- GitHub Pages para deploy

## Características

- Búsqueda de cualquier ciudad del mundo
- Temperatura, humedad, viento, sensación térmica y nubosidad
- Íconos dinámicos según el estado del clima
- Manejo de errores (ciudad no encontrada, sin conexión)
- Búsquedas rápidas de ciudades populares
- Diseño responsive (mobile y desktop)
